# DatoClaro - Sitio Institucional (One Page)

Sitio web institucional de **DatoClaro** (HTML + CSS + JS puro).  
Diseño minimalista, corporativo, responsive, con animaciones sutiles y SEO básico.

## Estructura del proyecto

- `index.html` → página principal (One Page)
- `assets/css/styles.css` → estilos (paleta, layout, responsive)
- `assets/js/main.js` → interacción (menú, animaciones, modal)
- `assets/img/` → imágenes (hero, ejemplos, favicon)
- `assets/docs/` → documentos (PDF ejemplo, opcional)

## Requisitos
No requiere dependencias ni build. Es estático.

## Cómo correrlo localmente
Opción A: abrir `index.html` con doble click.  
Opción B (recomendado): levantar servidor local.

### Con VS Code
1. Instalá la extensión **Live Server**
2. Click derecho en `index.html` → **Open with Live Server**

### Con Python
```bash
python -m http.server 8000
